# Учет успеваемости студентов

Простая консольная программа на языке Go, предназначенная для ведения электронного журнала: добавление студентов, ввод оценок, расчёт среднего балла, фильтрация и сортировка по успеваемости. Поддерживаются оценки от 1 до 5.

## Возможности

- Добавление студента по фамилии, имени и отчеству  
- Добавление оценок (только целые числа от 1 до 5)  
- Расчёт среднего балла для каждого студента  
- Вывод списка всех студентов с оценками и средним баллом  
- Фильтрация студентов с баллом ниже заданного порогового значения  
- Сортировка студентов по среднему баллу:
  - по убыванию
  - по возрастанию  

## Запуск

Для запуска программы необходимо:

1. Установить Go (версия 1.16 или выше)  
2. Сохранить исходный код в файл `main.go`  
3. Выполнить команду в терминале:
go run main.go

## Описание работы программы

Программа начинает выполнение с функции main(). Выводится заголовок и запускается бесконечный цикл меню, в котором пользователю предлагается выбрать одну из доступных операций:

1. Добавить студента
2. Добавить оценку студенту
3. Показать всех студентов
4. Показать студентов с баллом ниже заданного
5. Сортировка по среднему баллу (по убыванию)
6. Сортировка по среднему баллу (по возрастанию)
7. Выход из программы
После выбора операции (1–6) программа выполняет соответствующие действия:

- При добавлении студента — запрашивает ФИО и присваивает уникальный числовой ID
- При добавлении оценки — выводит список студентов, запрашивает ID и корректную оценку (1–5)
- При фильтрации и сортировке — запрашивает параметры (пороговое значение, направление сортировки) и отображает результат
- После завершения операции меню отображается снова. Программа завершает работу при выборе пункта 7.

## Описание функций

- `main()`:
  Точка входа в программу. Содержит основной цикл взаимодействия с пользователем, обрабатывает ввод, вызывает методы структуры Gradebook.

- `Student` (структура):
  Хранит данные студента: ID, ФИО, список оценок.
  
### Методы:

- `FullName()` — возвращает ФИО в формате «Фамилия Имя Отчество»
- `AddGrade(grade int)` — добавляет оценку (проверяет диапазон 1–5)
- `AverageGrade()` — возвращает средний балл (0.0 при отсутствии оценок)
- `Gradebook (структура)`:
  Представляет журнал успеваемости. Хранит студентов в map[int]*Student, генерирует уникальные ID.
  
### Методы:

- `NewGradebook()` — конструктор, создаёт пустой журнал
- `AddStudent(name, surname, patronymic string) int` — добавляет студента, возвращает его ID
- `AddGradeToStudent(id int, grade int)` — добавляет оценку студенту по ID (выводит ошибку, если студент не найден)
- `PrintAllStudents()` — выводит список всех студентов с оценками и средним баллом
- `StudentsBelowGrade(threshold float64) []*Student` — возвращает список студентов со средним баллом ниже заданного порога
- `SortByAverage(ascending bool) []*Student` — возвращает отсортированный по среднему баллу список студентов (true — по возрастанию, false — по убыванию)

## Особенности реализации

Все данные хранятся в оперативной памяти и теряются при завершении программы
ID студентов генерируются последовательно, начиная с 1, и не повторяются в рамках одной сессии
Ввод проверяется на корректность: при ошибках ввода отображается сообщение, операция прерывается, управление возвращается в меню
Средний балл выводится с точностью до двух знаков после запятой
